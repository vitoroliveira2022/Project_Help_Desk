npm create vite@latest -> inicia a criação de um projeto Vite

npm install -> instala as dependências definidas no projeto, ou seja, baixa tudo que o Vite precisa

npm run dev -> inicia o servidor de desenvolvimento.

npm install react-router-dom

npm install tailwindcss @tailwindcss/vite -> instalar Tailwind CSS

adapta o 'vite.config.js'
